Body of Work

Animal Shelter

Simple animal shelter webpage to practice div styling using css and html. Homepage offers sidebar with ability to link to other parts of website. Images, names, and descriptions of animals are arranged into columns. Using html and css, the columns are formatted to rearrange themselves, according to the width of the webpage in pixels.

Wiki

Webpage designed to imitate the look of wikipedia's homepage. Using bootsrap rows and columns, the page is organized into sections similar to those on wikipedia. Each section of wikipedia was inspected to determine its exact color, which was then applied to our sections in our css stylesheet. The sidebar and the main page are in different containers, each set to a specific width and floated to the left and right, so as to not overlap.

Cupcake

Intended to practice creating classes and ids in html and formatting in css. This webpage includes basic html elements such as images, hyperlinks, lists, paragraphs, and headers.
